## 系统简介

理财配置回测系统是一个专业的 Web 版投资分析工具，支持对基金产品进行历史回测分析。系统采用苹果风格设计，界面简洁美观，操作直观便捷。

### 主要功能

- **产品管理**：添加、编辑、删除基金产品，同步历史数据
- **智能配置**：基于风险承受能力自动生成资产配置建议
- **回测中心**：创建回测方案，配置资产组合，查看回测结果
- **数据分析**：方案对比、蒙特卡洛模拟、相关性分析
- **一键启停**：通过脚本快速启动和停止前后端服务

### 支持的理财产品

| 产品类型 | 说明 | 特点 |
|---------|------|------|
| 纳指ETF | 纳斯达克100指数基金 | 美股科技龙头，高成长 |
| 标普ETF | 标普500指数基金 | 美股大盘，稳健增长 |
| 混债基金 | 偏债混合型基金 | 股债平衡，稳健收益 |
| 红利低波ETF | 红利低波动指数基金 | 高分红，低波动 |
| 中证A50ETF | 中证A50指数基金 | A股核心资产 |

## 环境要求

## 系统启动

### 方式一：一键启动（推荐）

系统提供了一键启停脚本，方便快捷地管理服务。

#### 启动所有服务

在项目根目录执行：

```powershell
.\start-all.ps1
```

或：

```powershell
.\manage.ps1 start
```

此命令会：
1. 自动启动后端 Flask 服务（http://localhost:5000）
2. 自动启动前端 Vue 服务（http://localhost:3000）
3. 自动打开浏览器访问系统

#### 启动选项

```powershell
# 仅启动后端
.\start-all.ps1 -BackendOnly

# 仅启动前端
.\start-all.ps1 -FrontendOnly

# 启动但不打开浏览器
.\start-all.ps1 -NoBrowser
```

#### 停止所有服务

```powershell
.\stop-all.ps1
```

或：

```powershell
.\manage.ps1 stop
```

#### 停止选项

```powershell
# 仅停止后端
.\stop-all.ps1 -BackendOnly

# 仅停止前端
.\stop-all.ps1 -FrontendOnly

# 强制停止（包括非脚本启动的进程）
.\stop-all.ps1 -Force
```

#### 查看服务状态

```powershell
.\manage.ps1 status
```

#### 重启服务

```powershell
.\manage.ps1 restart
```

### 方式二：手动启动

如需手动控制服务，可以分别启动前后端。

#### 启动后端服务

1. 打开终端，进入后端目录

```bash
cd backend
```

2. 安装依赖（首次使用）

```bash
pip install -r requirements.txt
```

3. 启动后端服务

```bash
python run.py
```

启动成功后，终端会显示：
```
Starting backend...
Python: 3.x.x
Working directory: D:\02 Codes\portfolio_backtest\backend
Creating app... success
Starting Flask server on http://0.0.0.0:5000
```

**后端服务地址**：http://localhost:5000

---

#### 启动前端服务

1. 打开新的终端，进入前端目录

```bash
cd frontend
```

2. 安装依赖（首次使用）

```bash
npm install
```

3. 启动前端开发服务器

```bash
npm run dev
```

启动成功后，终端会显示：
```
VITE v5.4.21  ready in xxx ms

➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
```

**前端访问地址**：http://localhost:3000

### 第三步：访问系统

打开浏览器，访问 http://localhost:3000 即可进入系统。

### 二、智能配置

基于您的风险承受能力，为您推荐最优资产配置方案。

#### 1. 进入智能配置

点击侧边栏"智能配置"菜单进入页面。

#### 2. 风险评估问卷

填写以下信息以评估您的风险承受能力：

| 评估维度 | 说明 | 选项示例 |
|---------|------|---------|
| 年龄 | 您的实际年龄 | 18-80岁滑块选择 |
| 投资经验 | 您的投资年限 | 1年以下、1-3年、3-5年等 |
| 年收入 | 年收入范围 | 10万以下、10-20万、20-50万等 |
| 流动资产 | 可投资资产规模 | 10万以下、10-50万、50-100万等 |
| 投资期限 | 计划投资时长 | 1年以内、1-3年、3-5年等 |
| 最大可承受损失 | 能接受的最大亏损 | 5%以内、5-10%、10-20%等 |
| 每月投资金额 | 计划每月投入金额 | 1000-1000000元 |
| 投资目标 | 主要投资目的 | 资产保值、稳定收益、平衡增长、资本增值、激进增长 |

#### 3. 获取配置建议

填写完问卷后，点击"获取智能配置建议"按钮，系统将根据您的回答：

1. **评估风险等级**：保守型、谨慎型、平衡型、进取型、激进型
2. **计算风险评分**：0-100分的风险承受能力评分
3. **推荐资产配置**：基于风险等级推荐最优配置方案

#### 4. 查看配置方案

评估结果页面展示：

- **风险等级徽章**：显示您的风险类型（彩色标识）
- **风险评分**：进度条展示评分
- **建议权益仓位**：推荐的股票类资产比例
- **资产配置饼图**：各类资产的配置比例可视化
- **配置逻辑说明**：解释推荐配置的原因

#### 5. 市场环境调整

根据当前市场环境，可调整配置：

- **牛市**：增加权益类资产配置，把握上涨机会
- **熊市**：增加防御性资产配置，降低组合波动
- **震荡市**：保持均衡配置，等待方向明确

#### 6. 应用到回测

点击"应用到回测"按钮，系统将：
1. 自动将推荐配置填充到回测配置页面
2. 跳转到回测配置页面
3. 您只需补充回测区间等参数即可开始回测

### 三、产品管理

用于管理系统中的基金产品信息。

#### 1. 查看产品列表

进入"产品管理"页面，可以看到所有已添加的产品：

| 字段 | 说明 |
|------|------|
| 产品代码 | 基金的唯一代码，如 513100 |
| 产品名称 | 基金的完整名称 |
| 类型 | ETF 或 基金 |
| 分类 | 产品的细分分类 |
| 管理费率 | 基金管理费用比例 |
| 操作 | 查看数据、编辑、删除 |

#### 2. 添加新产品

1. 点击右上角的"添加产品"按钮
2. 填写产品信息：
   - **产品代码**：必填，如 513100
   - **产品名称**：必填，如 国泰纳斯达克100ETF
   - **产品类型**：选择 ETF 或 基金
   - **分类**：可选，如 海外指数
   - **管理费率**：可选，如 0.0060 表示 0.6%
   - **申购费率**：可选
   - **赎回费率**：可选
   - **描述**：可选，产品的详细说明
3. 点击"确定"保存

#### 3. 编辑产品

1. 在产品列表中找到要编辑的产品
2. 点击"编辑"按钮
3. 修改相关信息（产品代码不可修改）
4. 点击"确定"保存

#### 4. 删除产品

1. 在产品列表中找到要删除的产品
2. 点击"删除"按钮
3. 确认删除操作

⚠️ **注意**：删除产品会同时删除相关的历史数据，请谨慎操作。

#### 5. 同步历史数据

1. 在产品列表中勾选需要同步的产品（可多选）
2. 点击"同步数据"按钮
3. 等待同步完成

或者查看单个产品的数据：
1. 点击产品行的"查看数据"按钮
2. 在弹出的对话框中查看历史数据
3. 点击"同步最新数据"获取最新数据

#### 6. 初始化默认产品

系统预设了5只常用基金产品，点击"初始化默认产品"可一键添加：
- 纳指ETF (513100)
- 标普ETF (513500)
- 混债基金 (000171)
- 红利低波 (515080)
- 中证A50 (560050)

### 四、回测中心

用于创建和管理投资组合回测方案。

#### 1. 查看回测列表

进入"回测中心"页面，可以看到所有已创建的回测方案：

- 方案名称
- 回测区间（开始日期 ~ 结束日期）
- 初始金额
- 再平衡策略
- 操作按钮（查看结果、删除）

#### 2. 创建回测方案

点击"新建回测"按钮，进入配置页面：

##### 2.1 基本信息

- **方案名称**：给回测方案起个名字，如"保守型配置"
- **回测区间**：选择回测的开始日期和结束日期
  - 建议至少选择1年以上的数据
  - 可以选择历史数据较完整的区间
- **描述**：可选，对方案的说明

##### 2.2 资金配置

- **初始金额**：回测的起始资金，默认 100,000 元
- **定投策略**：
  - **不定投**：一次性投入初始金额
  - **定期定额**：每月固定金额定投
- **定投金额**：选择定期定额时填写，如 5,000 元/月

##### 2.3 再平衡策略

- **不调整**：买入后一直持有，不做调整
- **定期再平衡**：按固定周期调整资产配置
  - **再平衡周期**：如 12 个月
- **阈值再平衡**：当资产偏离目标配置一定比例时调整
  - **偏离阈值**：如 10%，表示偏离超过10%时触发再平衡

##### 2.4 资产配置

配置各产品的投资比例，总和必须等于 100%。

**方式一：使用智能配置（推荐）**

1. 点击"智能配置建议"按钮
2. 在智能配置页面完成风险评估
3. 获取推荐配置方案
4. 点击"应用到回测"自动填充配置

**方式二：手动配置**

1. 点击"添加配置"添加一行
2. 选择产品（从已添加的产品中选择）
3. 拖动滑块或输入数值设置比例
4. 重复以上步骤添加多个产品
5. 右上角显示"总计"，必须为 100%

**示例配置**：
- 纳指ETF：40%
- 标普ETF：20%
- 混债基金：20%
- 红利低波：10%
- 中证A50：10%

配置完成后，点击"开始回测"按钮执行回测。

#### 3. 查看回测结果

在回测列表中点击方案，或点击"查看结果"按钮，进入结果页面：

##### 3.1 关键指标

| 指标 | 说明 | 解读 |
|------|------|------|
| 年化收益率 | 投资在一年内的收益率 | 越高越好 |
| 年化波动率 | 收益率的标准差 | 衡量风险，越低越稳定 |
| 最大回撤 | 从峰值到谷底的最大跌幅 | 越低越好 |
| 夏普比率 | 风险调整后的收益指标 | >1 较好，>2 优秀 |

##### 3.2 净值曲线

展示投资组合的净值变化趋势：
- 横轴：日期
- 纵轴：净值
- 曲线：组合净值走势

##### 3.3 持仓明细

展示回测期间的持仓变化情况：
- 日期
- 产品代码
- 份额
- 市值
- 权重

#### 4. 删除回测方案

在回测列表中点击"删除"按钮，确认后删除方案。

### 五、数据分析

提供多种分析工具，帮助深入理解投资组合。

#### 1. 方案对比

对比多个回测方案的表现：

1. 进入"数据分析"页面，选择"方案对比"标签
2. 在"选择要对比的方案"下拉框中选择至少2个方案
3. 点击"开始对比"按钮
4. 查看对比图表和指标

对比内容包括：
- 各方案的净值曲线对比
- 风险收益指标对比

#### 2. 蒙特卡洛模拟

通过随机模拟预测投资组合的未来表现：

1. 选择"蒙特卡洛模拟"标签
2. 选择要模拟的方案
3. 设置模拟参数：
   - **模拟次数**：建议 1000-5000 次
   - **预测年限**：建议 5-10 年
4. 点击"运行模拟"按钮
5. 查看模拟结果：
   - 中位数路径
   - 95% 置信区间（上限和下限）

**解读**：
- 中位数代表最可能的结果
- 置信区间展示了结果的不确定性范围
- 区间越窄，预测越确定

#### 3. 相关性分析

分析不同产品之间的相关性：

1. 选择"相关性分析"标签
2. 选择至少2个产品（建议3-5个）
3. 点击"分析相关性"按钮
4. 查看相关性热力图

**解读**：
- 相关系数范围：-1 到 1
- 1：完全正相关（同涨同跌）
- 0：无相关性
- -1：完全负相关（此消彼长）
- 构建组合时，选择相关性较低的产品可以分散风险

## 常见问题

### Q1: 系统无法启动怎么办？

**后端无法启动**：
1. 检查 Python 版本是否为 3.8+
2. 检查依赖是否安装完整：`pip list`
3. 检查端口 5000 是否被占用
4. 查看错误日志，根据提示解决问题

**前端无法启动**：
1. 检查 Node.js 版本是否为 18+
2. 检查依赖是否安装完整：`npm list`
3. 检查端口 3000/3001 是否被占用
4. 尝试删除 `node_modules` 重新安装

### Q2: 无法获取基金数据怎么办？

1. 检查网络连接是否正常
2. 确认产品代码是否正确
3. 部分产品可能数据源不支持，尝试其他产品
4. 查看后端日志了解具体错误

### Q3: 回测结果不准确？

1. 检查回测区间是否有足够的历史数据
2. 确认资产配置比例总和为 100%
3. 检查是否考虑了费率影响
4. 注意回测基于历史数据，不代表未来表现

### Q4: 如何备份数据？

系统数据存储在 `data/portfolio.db` 文件中：
1. 定期复制该文件进行备份
2. 需要恢复时替换该文件即可

### Q5: 可以添加自定义产品吗？

可以，在"产品管理"页面点击"添加产品"，填写产品信息即可。需要确保：
- 产品代码正确
- 数据源支持该产品

### Q6: 一键启动脚本无法运行？

**PowerShell 执行策略限制**：
1. 以管理员身份运行 PowerShell
2. 执行命令：`Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`
3. 输入 `Y` 确认

**脚本闪退或无响应**：
1. 检查是否正确安装了 Python 和 Node.js
2. 检查端口 5000 和 3000 是否被占用
3. 尝试使用 `.\manage.ps1 status` 查看服务状态
4. 使用 `.\stop-all.ps1 -Force` 强制清理残留进程

### Q7: 智能配置建议不准确？

1. 确保风险评估问卷填写真实准确
2. 智能配置基于历史数据和算法模型，仅供参考
3. 建议结合自身情况和市场环境综合判断
4. 可使用"市场环境调整"功能微调配置
